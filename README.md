# Top Marketing - Modern Corporate Website

A professional, high-performance corporate website redesign for **Top Marketing**, featuring a bilingual interface (Polish/English), modern UI components, and a fully responsive design.

## 🚀 Live Demo
View the live project here: [https://styxinthewastelands.github.io/topmarketing-redesign](https://styxinthewastelands.github.io/topmarketing-redesign)

---

## ✨ Features

*   **Bilingual Support:** Full translation between Polish and English using a custom React Context-based localization system.
*   **Modern UI/UX:** Built with a "Mobile First" approach using Tailwind CSS for sleek, professional aesthetics.
*   **Dynamic Routing:** Implemented with `react-router` (HashRouter) for seamless navigation and GitHub Pages compatibility.
*   **Comprehensive Offerings:** Dedicated pages for:
    *   Gift Sets Production
    *   Co-packing & Packaging services
    *   Promotional Sweets
    *   Printing & Promotional Materials
*   **Interactive Elements:** Integrated with Lucide React icons and Framer Motion for subtle, professional animations.

---

## 🛠️ Tech Stack

*   **Framework:** [React 18](https://reactjs.org/)
*   **Build Tool:** [Vite](https://vitejs.dev/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Routing:** [React Router v7](https://reactrouter.com/)
*   **Deployment:** [GitHub Pages](https://pages.github.com/)

## 📦 Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   git clone [https://github.com/styxinthewastelands/topmarketing-redesign.git](https://github.com/styxinthewastelands/topmarketing-redesign.git)
2. **Navigate to the project folder:**
  cd topmarketing-redesign
3. **Install dependencies:**
   npm install
4. **Start the development server:**
   npm run dev
   Open http://localhost:5173/topmarketing-redesign in your browser.

---

## 🚀 Deployment

The project is configured for automatic deployment to GitHub Pages.
To deploy the latest version:
  npm run deploy
